# Submarines-game
The classic game of submarines. Text based UI.
## Game features
* **Difficulty:** Difficulty settings affecting enemy "luck".
* **Basic AI capabilities:** A guessing strategy for the enemy based on a succesful previous shot.
* **Vessel placement:** 4 unrelated board presets for each player, randomly chosen.

## Running the game
* The game runs as a C program, so you will need to compile and run it on your own device's terminal.
* Aside from cloning and compiling the program, no additional preparation is needed to play.

## How to play
* The game operates on a 6x6 coordinate grid.
* Each turn, enter your target row (vertical axis) and column (horizontal axis) to fire.
* Basic game instruction are provided in the games start up menu.
* The first player to sink all enemy vessels wins.
